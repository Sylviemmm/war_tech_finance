# 战争技术金融事件可视化平台
**项目全程AI生成，使用GLM-4.7模型**

## 项目概述
基于腾讯地图API的战争、技术、金融历史事件可视化分析平台，展示1837-2020年间的重大历史事件及其关联关系。

## 功能特性

### 🗺️ 地图可视化
- 腾讯地图API集成，支持全球事件地理分布展示
- 不同颜色标识事件类型：战争(红色)、技术(黄色)、金融(蓝色)、关系(粉色)
- 动态散点图层，支持辐射动画效果
- 点击事件标记显示详细信息

### 📊 多维度数据展示
- **战争事件**: 10场重大战争，从克里米亚战争到俄乌冲突
- **技术事件**: 22项关键技术突破，从电报到人工智能
- **金融事件**: 16个重要金融事件，从战争债券到新冠疫情冲击
- **关联关系**: 15个事件间的关联关系，展示战争、技术、金融的相互影响

### 🔍 强大的筛选功能
- 按事件类型筛选
- 按影响程度筛选
- 按时间范围筛选
- 实时更新统计数据

### 📱 三种视图模式
- **概览**: 统计数据和筛选控制
- **时间线**: 按时间顺序展示所有事件
- **关系**: 展示事件之间的关联关系

## 快速开始

### 前置要求
- 现代浏览器（Chrome、Edge、Firefox等）
- 腾讯地图API密钥

### 使用方法
1. 直接在浏览器中打开 `index.html`
2. 如需使用自己的腾讯地图API密钥，修改代码中的 `key` 参数

## 数据说明

### 数据来源
- **战争数据**: UCDP/PRIO数据库
- **技术数据**: 专利数据库、军事史文献、航空史文献等
- **金融数据**: 金融史文献、经济史文献等
- **关系数据**: 关系数据库

### 数据格式
- CSV格式，包含事件ID、类型、名称、时间、地点、参与方、描述、影响程度等字段
- 支持事件间的关联关系（related_war_id、related_tech_id、related_finance_id）

## 核心功能

### 事件类型
- **战争**: 国际冲突和军事行动
- **技术**: 重大技术突破和创新
- **金融**: 重要金融事件和政策变化
- **关系**: 事件间的关联和影响

### 影响程度
- **极高**: 对全球产生重大影响的事件
- **高**: 对地区或领域产生重要影响的事件
- **中**: 对特定方面产生一定影响的事件

### 交互功能
- 点击地图标记查看事件详情
- 点击时间线事件定位到地图
- 筛选器实时更新显示内容
- 支持地图缩放和平移
- 可切换侧边栏显示
