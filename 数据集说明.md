# 战争·技术·金融可视化分析平台 - 数据集说明

## 数据集概述

本项目包含三个核心CSV数据集文件，用于支撑战争、技术与金融关系的可视化分析研究。

## 数据集文件列表

### 1. war_tech_finance_dataset.csv
**综合事件数据集**

包含战争、技术、金融事件以及它们之间的关系数据。

**主要字段：**
- `event_id`: 事件唯一标识符
- `event_type`: 事件类型（war/technology/finance/relation）
- `event_name`: 事件名称
- `start_date`: 开始日期
- `end_date`: 结束日期
- `location`: 地理位置
- `parties_involved`: 参与方
- `description`: 事件描述
- `impact_level`: 影响程度（low/medium/high/very_high）
- `related_war_id`: 关联战争ID
- `related_tech_id`: 关联技术ID
- `related_finance_id`: 关联金融事件ID
- `data_source`: 数据来源

**数据规模：** 约60条记录

### 2. technology_timeline_dataset.csv
**技术时间线数据集**

详细记录技术发明的三个关键时间节点：发明时间、投资时间、普及时间。

**主要字段：**
- `tech_id`: 技术唯一标识符
- `tech_name`: 技术名称
- `tech_category`: 技术类别
- `invention_date`: 发明时间
- `invention_country`: 发明国家
- `inventor`: 发明者
- `investment_start_date`: 投资开始时间
- `investment_country`: 投资国家
- `investment_amount_usd`: 投资金额（美元）
- `popularization_date`: 普及时间
- `popularization_scope`: 普及范围
- `military_application_date`: 军事应用时间
- `civilian_application_date`: 民用应用时间
- `war_relation`: 与战争的关系
- `finance_relation`: 与金融的关系
- `impact_description`: 影响描述
- `data_source`: 数据来源

**数据规模：** 30条记录

**时间维度分析：**
- **发明时间**：技术原理首次验证或专利授予时间
- **投资时间**：国家或资本大规模投入时间
- **普及时间**：技术广泛应用或成熟应用时间

### 3. war_finance_impact_dataset.csv
**战争金融影响数据集**

详细记录战争对金融市场的具体影响数据。

**主要字段：**
- `war_id`: 战争唯一标识符
- `war_name`: 战争名称
- `war_start`: 战争开始时间
- `war_end`: 战争结束时间
- `war_type`: 战争类型
- `war_cause`: 战争原因
- `total_casualties`: 总伤亡人数
- `total_cost_usd`: 总成本（美元）
- `gdp_impact_percent`: GDP影响百分比
- `stock_market_impact`: 股市影响百分比
- `war_bond_amount`: 战争债券金额
- `currency_devaluation_percent`: 货币贬值百分比
- `inflation_rate_peak`: 通胀率峰值
- `post_war_recovery_years`: 战后恢复年数
- `major_financial_innovation`: 主要金融创新
- `economic_consequences`: 经济后果
- `long_term_economic_impact`: 长期经济影响
- `data_source`: 数据来源

**数据规模：** 20条记录

## 数据特点

### 时间跨度
- **起始时间**：1850年（克里米亚战争）
- **结束时间**：2024年（俄乌冲突进行中）
- **时间跨度**：约174年

### 地理覆盖
- 欧洲、亚洲、非洲、美洲
- 主要涉及国家：英国、法国、德国、美国、俄罗斯、中国等

### 数据质量
- 数据来源：UCDP/PRIO、专利数据库、金融史文献、军事史文献等权威来源
- 数据准确性：基于历史记录和学术研究
- 数据完整性：关键字段完整率>95%

## 数据关系

### 实体关系
1. **战争 → 技术**：战争催生技术发展（如二战催生雷达、原子弹）
2. **技术 → 战争**：技术应用于战争（如坦克改变陆战形态）
3. **战争 → 金融**：战争影响金融市场（如战争债券、股市波动）
4. **金融 → 技术**：金融投资技术发展（如风险投资互联网）
5. **金融 → 战争**：金融支持战争进行（如战争融资）

### 时间关系
- 战争爆发时间与技术发明时间的关联
- 技术投资时间与军事应用时间的关联
- 金融事件发生时间与战争周期的关联

## 使用建议

### 可视化应用
1. **时间轴可视化**：使用`start_date`和`end_date`字段
2. **地理空间可视化**：使用`location`和`invention_country`字段
3. **关系网络可视化**：使用`related_*_id`字段建立关联
4. **统计分析**：使用数值型字段进行相关性分析

### 数据分析
1. **时间序列分析**：分析战争、技术、金融事件的时间分布
2. **相关性分析**：计算战争频率与技术发明数量的相关性
3. **因果分析**：识别战争对技术发展和金融市场的影响
4. **比较分析**：对比不同战争类型和不同国家的差异

### 数据扩展
1. 可以添加更多战争事件，特别是地区性冲突
2. 可以补充更多技术发明，特别是民用技术
3. 可以添加更多金融事件，特别是股市和汇率数据
4. 可以补充更多关系数据，建立更完整的知识图谱

## 数据更新

### 更新频率
- 建议每季度更新一次

### 更新内容
1. 补充新的战争事件
2. 更新进行中的战争数据
3. 添加新兴技术（如AI、量子计算）
4. 补充最新的金融事件

## 注意事项

1. **数据准确性**：部分历史数据存在争议，使用时需注明数据来源
2. **货币换算**：历史货币金额已换算为当前美元价值
3. **伤亡统计**：不同来源的伤亡数据可能存在差异
4. **时间精度**：部分历史事件的具体日期可能存在不确定性

## 引用格式

使用本数据集时，请引用以下格式：

```
战争·技术·金融可视化分析平台数据集（2024）
数据来源：UCDP/PRIO、专利数据库、金融史文献、军事史文献
```
